# Calculator
## Overview:
### Goals and Specifications:
1- Mathematical operations

2- Quadratic equation

3- Array Operations

### Mathematical operations: 
In this part, the program performs one of mathematical operations like 
(addition, subtraction, multiplication, or division) 
on any two number or more. 

### Quadratic equation:
In this part, the program is responsible to solve a quadratic equation. 
It prompts a user to enter coefficients a, b, and c of the quadratic equation ax2 + bx + c = 0. 
Once the user entered the coefficient values, the program produces the related answers of x1 
and x2.

### Array Operations:
In this part, the program is responsible to enable a user to add, sub, or multiply two arrays. The program prompts the user to enter the array operations: addition, subtraction, multiplication,show matrix or get value. Once the user entered the required operation, he will be asked again to enter the array dimensions. After entering the dimensions, he will be asked to enter each element of the two arrays. Finally, the program produces or prints the related results in a  form of array.
